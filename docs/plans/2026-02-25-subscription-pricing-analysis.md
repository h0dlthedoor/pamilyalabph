## Expected Value Analysis: Three Price Points

### Assumptions

- Kris's direct unit: ~15 people
- Kris has personal relationships = high trust = higher conversion than cold sales
- Agents are commission-based and price-sensitive

### Scenario A: PHP 199/month

| Outcome | Probability | Subscribers | Monthly Rev | Annual Rev |

| Moderate adoption | 50% | 8 | PHP 1,592 | PHP 19,104 |

### EV Comparison

| Price Point | Monthly EV | Annual EV | Adoption Risk | Anchoring Effect |
|-------------|-----------|-----------|---------------|------------------|
| PHP 199 | PHP 1,632 | PHP 19,582 | Low risk, but leaves money on table | Anchors LOW — hard to raise later |
| **PHP 299** | **PHP 2,018** | **PHP 24,219** | **Sweet spot — matches Canva Pro** | **Matches existing mental model** |
| PHP 499 | PHP 1,622 | PHP 19,461 | High friction, 40% chance only 1 sub | Perceived as "expensive tool" |

**Winner: PHP 299/month — highest expected value by ~24% over PHP 199.**

---

## Why PHP 199 is Too Low

1. **Price signals value.** At PHP 199, cheaper than a McDonald's family meal. Signals "toy" not "professional tool."
2. **Anchoring problem.** Once set at 199, very hard to raise later. Starting at 299 with a "launch discount" of 199 for 3 months is far better.
3. **Canva Pro comparison.** PH agents already pay PHP 300/month for Canva Pro — a graphics tool. PamilyaLab delivers MORE direct revenue value (lead generation + needs analysis + branded microsite). Charging LESS than Canva signals less value.
4. **The "coffee money" trap.** 8 subscribers x PHP 199 = PHP 1,592/month. At 7 subscribers x PHP 299 = PHP 2,093/month — 31% more revenue with FEWER subscribers.

## Why PHP 499 is Too High

1. **Crosses pain threshold.** PHP 499 = PHP 5,988/year, roughly equal to monthly Facebook ads budget. Agents compare directly to running ads.
2. **Only ~10-20% of agents pay for ANY tool.** PHP 499 filters out the majority.
3. **Small team context.** For 5-20 people, maximizing ADOPTION matters more than per-user revenue.

## Recommended Pricing Strategy

```
┌─────────────────────────────────────────────────┐
│           RECOMMENDED: PHP 299/month            │
│                                                 │
│  Launch offer: PHP 199/month for first 3 months │
│  Then auto-renews at PHP 299/month              │
│                                                 │
│  Annual plan: PHP 2,988/year (PHP 249/mo)       │
│  = 2 months free vs monthly billing             │
└─────────────────────────────────────────────────┘
```

| Tactic | Effect |
|--------|--------|
| PHP 199 launch price | Removes friction for first adopters, feels risk-free |
| 3-month trial period | Enough time for agents to capture real leads and see value |
| PHP 299 standard price | Anchored at Canva Pro — a price agents already accept |
| Annual discount (PHP 249/mo) | Locks in committed users, improves cash flow for Kris |
| "2 months free" framing | More compelling than "17% discount" |
